# Free-Parallel-AEO
Opensource AEO multi LLM architecture designed to help business and people optimize their AEO. The idea behind this project arose when at the company I currently worked for we started assessing our AEO and a lot of paid tools and SaaS was offered to us at low prices but still why pay for it when the whole architecture is relatively simple. I build this "repo" to help us assess our AEO internally and am now sharing it with anyone who does not want to pay for such tools for no apparent reason.

## Fundation
The tool as mainly based on OpenRouter and its abaility to operate SOTA LLMs in parallel from the same simple script. 

## Pre-requesites
- An OpenRouter key - this is free but a run of SOTA models might cost anywhere from $0.50 to $5 per run without any major modicitations
- Python (any)
That is it to get you started, but optional:
- Claude subscription

## "Agentic" execution
So as to fit the word agentic in this repo I will also add a skill file or something to get this whole system to be fully self-owned by your own LLM of choice.
